# Upstart Landing Page

A simple startup product landing page built using React.js.

## About the Project
Upstart is a demo platform designed to help early-stage startups organize tasks,
track progress, and understand their workflow from one place.

## Tech Stack
- React.js
- Vite
- CSS
- Deployed on Vercel
